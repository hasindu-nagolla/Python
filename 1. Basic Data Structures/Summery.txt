list = [1, 2, 3, 4, 5]                                     # list
dictionary = {'key1': 'value1', 'key2': 'value2'}          # dictionary
set = {1, 2, 3, 4, 5}                                      # set
tuple = (1, 2, 3, 4, 5)                                    # tuple

(1) A list is used to store multiple items in a single variable.

(2)Dictionary is used to store data values in key:value pairs.

(3) A set is a collection of unordered elements. It is mutable and has no duplicate elements.

(4) A tuple is a collection of items which is ordered and unchangeable. Allows duplicate members.
